# EduPlay - Gamified Learning PWA

Современное мобильное веб-приложение (PWA) для образовательной геймификации. Превращает процесс обучения в интерактивную игру с системой очков, уровней и таблицей лидеров.

## 🎯 Особенности

- ✅ **Progressive Web App** - Установка на главный экран смартфона
- ✅ **Mobile-First Design** - Оптимизировано для мобильных устройств
- ✅ **Темная тема** - Современный темный интерфейс с анимациями
- ✅ **Игровая система** - XP, уровни, очки и серии
- ✅ **Аккаунты** - Регистрация, вход, авто-логин, профиль и выход
- ✅ **Аватары** - 16 стильных пресетов с возможностью смены
- ✅ **Достижения** - Награды с анимированными уведомлениями
- ✅ **Таймер вопроса** - Ограничение времени на ответ
- ✅ **Таблица лидеров** - localStorage для хранения результатов
- ✅ **Выбор режима перед игрой** - Категория + уровень сложности
- ✅ **Большой банк вопросов** - 126 вопросов (по 42 на категорию)
- ✅ **Плавные анимации** - Современный UX с переходами

## 🛠️ Технологии

- **React 18** - UI библиотека
- **TypeScript** - Типизированный JavaScript
- **Tailwind CSS** - Утилитарный CSS фреймворк
- **Vite** - Быстрый build tool
- **PWA** - Service Worker для офлайн работы

## 📂 Структура проекта

```
src/
├── components/        # Переиспользуемые компоненты (Button, Card, Header и т.д.)
├── pages/            # Экраны приложения (Quiz, Results, Leaderboard)
├── hooks/            # Пользовательские React хуки (useQuiz, useTimer)

├── App.tsx           # Главный компонент приложения
├── main.tsx          # Входная точка
└── index.css         # Глобальные стили
```

## 🚀 Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Шаги установки

```bash
# 1. Перейдите в директорию проекта
cd "STEM проект EduPlay"

# 2. Установите зависимости
npm install

# 3. Запустите development сервер
npm run dev

# 4. Откройте http://localhost:5173 в браузере
```

### Production сборка

```bash
# Создать оптимизированную сборку
npm run build

# Просмотр сборки локально
npm run preview
```

## 📱 Установка как PWA

### На iOS
1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран Домой"

### На Android
1. Откройте приложение в Chrome
2. Нажмите меню (три точки)
3. Выберите "Установить приложение"

## 🎮 Как использовать

1. **Главное меню** - Видите ваш уровень, очки и XP
2. **Перед стартом** - Выберите категорию и сложность (или сбалансированный режим)
3. **Начать викторину** - Ответьте на 10 случайных вопросов
4. **Система очков (скорость ответа):**
   - ✅ Easy: до +5, Medium: до +7, Hard: до +10
   - ❌ Ошибки: Easy -10, Medium -15, Hard -20
   - ⚡ Чем быстрее ответ, тем больше очков
   - 🛡️ Анти-рандом: доп. штраф за очень быстрые ошибки и серии ошибок
5. **Уровни** - Каждые 100 XP = новый уровень
6. **Таблица лидеров** - Сохраняйте свои результаты и конкурируйте

## 📊 Вопросы

В приложении встроено 126 вопросов школьного уровня:
- История Казахстана — 42
- Английский язык — 42
- Информатика — 42

В каждой категории вопросы разделены на Easy / Medium / Hard и выбираются случайно.

Вы можете легко добавить свои вопросы в `src/data/questions.ts`.

### Добавление новых компонентов

Все компоненты в `src/components/` следуют одному паттерну:
- TypeScript interfaces для props
- Единственная ответственность
- Переиспользуемость
- Tailwind для стилизации

## 💾 Данные

Все данные сохраняются в `localStorage`:
- **Аккаунты пользователей** - username, email, аватар, статистика
- **Игровой прогресс** - Очки, уровень, XP, серия, точность, лучшие результаты
- **Достижения** - Открытые награды и время получения
- **Таблица лидеров** - Top 50 игроков
