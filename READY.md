## 📱 Установка на iPhone (3 шага):

### Шаг 1: Откройте Firewall ⚙️
**Способ A (быстрый):**
```powershell
# Откройте PowerShell как администратор (Win + X)
New-NetFirewallRule -DisplayName "EduPlay" -Direction Inbound -Action Allow -Protocol TCP -LocalPort 5173 -Profile Any
```

**Способ B (через UI):**
- Win + R → `wf.msc` → Правила для входящих подключений → Создать правило → TCP порт 5173 → Разрешить

Подробнее: откройте файл **`FIREWALL_FIX.md`**

### Шаг 2: На iPhone откройте Safari 📱
1. Safari → адресная строка
2. Введите: `http://192.168.100.7:5173/`
3. Нажмите Go

### Шаг 3: Установите приложение 🏠
1. Нажмите **Поделиться** (стрелка вверх)
2. Выберите **"На экран Домой"**
3. Нажмите **"Добавить"**

**Готово! 🎉 Приложение установлено как нативное на iPhone!**
