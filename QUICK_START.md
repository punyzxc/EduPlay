# 🚀 EduPlay - Быстрый Старт

## ⚡ Как запустить приложение

### Шаг 1: Установить зависимости

```bash
# Откройте PowerShell в директории проекта
cd "c:\Users\puny\Desktop\STEM проект EduPlay"

# Добавьте Node.js в PATH
$env:PATH += ';C:\Program Files\nodejs'

# Установите зависимости
npm install
```

### Шаг 2: Запустить Dev Сервер

```bash
# Добавьте Node.js в PATH (если нужно)
$env:PATH += ';C:\Program Files\nodejs'

# Запустите приложение
npm run dev
```

Сервер запустится на **http://localhost:5173/**

### Шаг 3: Откройте в браузере

```
http://localhost:5173/
```

## 🎮 Как использовать приложение

1. **Главный экран** - Видите ваш уровень, очки и XP
2. **Нажмите "🚀 Начать Викторину"** - Начните отвечать на вопросы
3. **Ответьте на все 10 вопросов** - Выбирайте правильный ответ за 30 секунд
4. **Получите результаты** - Посмотрите сколько вы набрали очков
5. **Проверьте таблицу лидеров** - Сохраните свой результат

## 📊 Система очков

- ✅ Правильный ответ на **easy** = **+10** очков
- ✅ Правильный ответ на **medium** = **+20** очков
- ✅ Правильный ответ на **hard** = **+30** очков
- ⏱️ Бонус за скорость = до **15** дополнительных очков
- ❌ Неправильный ответ = **-5** очков

## 📱 Установка как PWA

### На Android:
1. Откройте приложение в Chrome
2. Нажмите меню (три точки)
3. Выберите "Установить приложение"

### На iOS:
1. Откройте в Safari
2. Нажмите "Поделиться"
3. Выберите "На экран Домой"
