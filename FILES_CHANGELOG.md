## 🚀 Как начать использовать

### Для игроков:
1. Откройте приложение
2. Пройдите викторину
3. Результат автоматически добавится в рейтинг
4. Откройте "Таблица лидеров" для просмотра

### Для разработчиков:
```typescript
import { getDailyLeaderboard, addDailyResult } from './utils/dailyLeaderboard';

// Получить рейтинг
const leaderboard = getDailyLeaderboard();

// Добавить результат
addDailyResult(150, "Мой никнейм");
```
