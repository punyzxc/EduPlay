# 📱 Инструкция: Как подключить мобильное устройство к EduPlay

## ❌ Если сервер недоступен - проверьте:

### 1️⃣ **Убедитесь что оба устройства в одной сети:**
- Компьютер: WiFi подключен к маршрутизатору
- iPhone: Подключен к тому же WiFi сети
- Проверьте что сеть одна и та же в Settings → Wi-Fi

### 2️⃣ **Откройте порт 5173 в Windows Firewall:**

#### Способ A: Вручную через UI (Самый простой)
1. Нажмите `Win + R`
2. Введите `wf.msc`
3. Нажмите Enter (откроется Windows Firewall)
4. В левой панели выберите **"Правила для входящих подключений"**
5. Нажмите **"Создать правило..."** (в правой панели)
6. Выберите **"Порт"** → Next
7. Выберите **"TCP"** и укажите порт **5173**
8. Выберите **"Разрешить подключение"** → Next
9. Отметьте все три варианта (Домен, Приватная, Общая)
10. Введите название: **EduPlay**
11. Нажмите **Готово**

#### Способ B: Через PowerShell (требует администратор)
```powershell
netsh advfirewall firewall add rule name="EduPlay" dir=in action=allow protocol=tcp localport=5173
```

### 3️⃣ **Попробуйте альтернативные адреса:**

На iPhone откройте Safari и вводите один из этих адресов:
- `http://192.168.100.7:5173/` (основной)
- `http://localhost:5173/` (если на локально)

### 4️⃣ **Проверьте IP адрес компьютера:**

На компьютере откройте PowerShell и выполните:
```powershell
ipconfig
```

Найдите строку **"IPv4-адрес"** - используйте этот адрес вместо 192.168.100.7

Пример вывода:
```
IPv4-адрес. . . . . . . . . . . . : 192.168.100.7
```

### 5️⃣ **Убедитесь что dev сервер работает:**

Должна быть активна консоль с текстом:
```
VITE v5.4.21 ready in 273 ms
➜ Network: http://192.168.100.7:5173/
```

### 6️⃣ **На iPhone в Safari:**

1. Откройте браузер Safari
2. В адресную строку введите: `http://192.168.100.7:5173/`
3. Нажмите Go

### 7️⃣ **Установка PWA на iPhone:**

Когда приложение загрузилось:
1. Нажмите иконку **"Поделиться"** (стрелка вверх)
2. Прокрутите вниз и выберите **"На экран Домой"**
3. Нажмите **"Добавить"**

---

## 🆘 Если ничего не помогает:

Попробуйте альтернативный способ через ngrok (создает публичный URL):

```bash
npm install -g ngrok

# В другом терминале:
ngrok http 5173

# Система выдаст URL типа: https://xxxx-xx-xxx-xxx.ngrok.io
```

Тогда используйте этот URL на iPhone вместо IP адреса.
