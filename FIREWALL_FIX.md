# 📱 ИНСТРУКЦИЯ: Как исправить Firewall для доступа с мобильного

## ⚠️ Проблема:
Сервер работает, но iPhone не может подключиться. Это блокирует **Windows Firewall**.

## ✅ Решение (выберите ОДИН способ):

---

## **СПОСОБ 1: Через PowerShell (1 минута)** ⭐ РЕКОМЕНДУЕТСЯ

1. **Нажмите Win + X** → выберите **"Windows PowerShell (администратор)"**
   - Если спросит - нажмите **Да**

2. **Скопируйте и вставьте эту команду:**
```powershell
New-NetFirewallRule -DisplayName "EduPlay" -Direction Inbound -Action Allow -Protocol TCP -LocalPort 5173 -Profile Any
```

3. **Нажмите Enter**

4. **Если вывод показал "True" - готово! ✓**

---

## **СПОСОБ 2: Через GUI Firewall (2 минуты)**

1. **Нажмите Win + R**
2. **Введите:** `wf.msc`
3. **Нажмите Enter**
4. **В левой панели:** Выберите **"Правила для входящих подключений"**
5. **В правой панели:** Нажмите **"Создать правило..."**
6. **Откроется окно:**
   - Выберите **"Порт"** → **Next**
   - Выберите **"TCP"** 
   - Выберите **"Определенные локальные порты"** 
   - Введите: **5173**
   - Нажмите **Next**
7. **Выберите "Разрешить подключение"** → **Next**
8. **Отметьте ВСЕ три варианта:**
   - ✓ Домен
   - ✓ Приватная   
   - ✓ Общая
   - Нажмите **Next**
9. **Введите имя:** `EduPlay`
10. **Нажмите Готово** ✓

---

## **СПОСОБ 3: Через скрипт батника**

1. **Откройте блокнот (Notepad)**
2. **Скопируйте:**
```batch
@echo off
netsh advfirewall firewall add rule name="EduPlay" dir=in action=allow protocol=tcp localport=5173 profile=any
pause
```
3. **Нажмите Ctrl+Shift+S** → сохраните как `firewall.bat`
4. **Нажмите правой кнопкой на файл** → **"Запустить от имени администратора"**

---

## ✅ Как проверить что правило создано:

Снова откройте PowerShell (администратор) и выполните:
```powershell
Get-NetFirewallRule -DisplayName "EduPlay"
```

Если вывод показал правило - всё работает! ✓

---

## 📱 После открытия Firewall:

1. **На iPhone откройте Safari**
2. **Введите URL:**
   ```
   http://192.168.100.7:5173/
   ```
3. **Нажмите Go** ✓
4. **Когда загрузится приложение:**
   - Нажмите **Поделиться** (стрелка вверх)
   - Выберите **"На экран Домой"**
   - Нажмите **"Добавить"**

---

## 🆘 Если ничего не помогает:

### Проверьте:
1. ✓ Оба устройства подключены к **ОДНОЙ И ТОЙ ЖЕ WiFi сети**
2. ✓ IP адрес верный: **192.168.100.7** (проверьте окно терминала с dev сервером)
3. ✓ Сервер все еще запущен (должна быть открыта консоль с кодом)

### IP может быть другим!
- Если не работает на 192.168.100.7, откройте консоль и посмотрите строку:
  ```
  ➜ Network: http://XXX.XXX.XXX.XXX:5173/
  ```
- Используйте ЭТОТ адрес на iPhone
