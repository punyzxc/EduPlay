# 🎯 Ежедневный Рейтинг - Быстрый старт

## 🚀 Что это?

Полнофункциональная система ежедневного рейтинга (топ-10 за 24 часа) для EduPlay. Создана для стимулирования конкуренции и азарта между игроками.

## ✨ Основные возможности

- **🏆 Топ-10 участников** по сумме очков за день
- **⏱️ Автоматический сброс** в полночь (UTC)
- **🔄 Реальное время** обновление каждые 5 секунд
- **💾 Накопление очков** - каждая попытка добавляется
- **🟢 Выделение текущего игрока** в рейтинге
- **🥇🥈🥉 Медали** для топ-3
- **📊 Красивый UI** с градиентами и анимациями

## 🎮 Как это работает

### Игровой цикл:
1. 🎯 Игрок проходит викторину (10 вопросов)
2. 📊 Получает очки в зависимости от ответов
3. ✅ Результат **автоматически** добавляется в ежедневный рейтинг
4. 🏆 На странице "Таблица лидеров" видит свою позицию
5. ⏱️ Может видеть время до следующего сброса рейтинга

### Автоматический сброс:
- Каждый день в **полночь UTC** рейтинг полностью очищается
- Все начинают с нулевых очков
- Новая конкуренция начинается

## 📱 Использование

### Для игроков:

1. **Открыть приложение** → перейти на "Таблица Лидеров"
2. **Видеть топ-10** за 24 часа
3. **Завершить викторину** → результат автоматически добавляется
4. **Установить имя** (опционально) → нажать "Сохранить в рейтинг"
5. **Конкурировать** → пройти еще викторину и попасть выше

### Для разработчиков:

```typescript
import {
  getDailyLeaderboard,
  addDailyResult,
  getCurrentPlayerRank,
  getTimeUntilReset,
} from './utils/dailyLeaderboard';

// Получить текущий рейтинг
const leaderboard = getDailyLeaderboard();

// Получить мою позицию
const myRank = getCurrentPlayerRank();

// Время до сброса
const time = getTimeUntilReset();
```

## 🎯 Ключевые функции API

| Функция | Описание |
|---------|---------|
| `getDailyLeaderboard()` | Получить топ-10 за 24 часа |
| `addDailyResult(score, name)` | Добавить результат в рейтинг |
| `getCurrentPlayerRank()` | Получить позицию текущего игрока |
| `getTimeUntilReset()` | Получить время до сброса |
| `getPlayerName()` | Получить имя игрока |
| `setPlayerName(name)` | Установить имя |
| `getOrCreatePlayerId()` | Получить/создать ID браузера |
| `getLeaderboardDateFormatted()` | Дата в красивом формате |
