# ⏰ Система Ежедневного Рейтинга

## 🎯 Общая концепция

Система ежедневного рейтинга создана для стимулирования конкуренции и азарта среди игроков. Каждый день рейтинг полностью обновляется, давая всем равные шансы на победу.

### Ключевые особенности:
- **Топ-10 участников** по сумме очков за 24 часа
- **Автоматический сброс** в полночь (UTC)
- **Обновление в реальном времени** - каждые 5 секунд
- **Накопление очков** - каждая попытка добавляет очки к дневному результату
- **Отслеживание попыток** - видно количество попыток и лучший результат

---

## 📊 Как это работает

### 1. Сохранение результатов
Когда игрок завершает викторину:
- Результат **автоматически добавляется** в ежедневный рейтинг
- Если это первая попытка игрока за день → создается новая запись
- Если игрок уже играл → очки добавляются к существующему результату

### 2. Группировка по игроку
Система отслеживает каждого уникального игрока по его `userId`:
- Каждый браузер получает уникальный ID при первом запуске
- ID сохраняется в `localStorage` для консистентности
- Можно установить имя, которое будет отображаться в рейтинге

### 3. Автоматический сброс
Рейтинг автоматически сбрасывается **в полночь UTC**:
- Проверка происходит при загрузке приложения
- Проверка также происходит при добавлении нового результата
- Гарантирует честное сравнение за одни сутки

### 4. Обновление в реальном времени
На странице таблицы лидеров:
- Рейтинг обновляется каждые 5 секунд
- Отображается отсчет времени до следующего сброса
- Текущий игрок выделен зеленым цветом

## 💾 Структура данных

### DailyLeaderboardData (хранится в localStorage)
```typescript
{
  date: "2024-04-22",              // YYYY-MM-DD текущей даты
  entries: [...],                  // Массив записей (макс 10)
  lastReset: 1713792000000        // Timestamp последнего сброса
}
```

### DailyLeaderboardEntry (каждая запись в рейтинге)
```typescript
{
  userId: "player_1234567890",     // Уникальный ID браузера
  name: "Игрок",                   // Имя для отображения
  totalScore: 250,                 // Сумма всех очков за день
  bestScore: 100,                  // Лучший результат одной попытки
  attemptsCount: 3,                // Количество завершённых попыток
  rank: 1,                         // Место в рейтинге (1-10)
  timestamp: 1713792000000,        // Когда был первый результат
  lastUpdate: 1713792300000        // Когда был последний результат
}
```

## 🔑 Ключевые функции API

### В `src/utils/dailyLeaderboard.ts`

#### `getDailyLeaderboard(): DailyLeaderboardEntry[]`
Получить текущий рейтинг топ-10 с автоматической проверкой сброса.

```typescript
const leaderboard = getDailyLeaderboard();
console.log(leaderboard[0]); // Первое место 🥇
```

#### `addDailyResult(score: number, playerName?: string): void`
Добавить результат в ежедневный рейтинг.

```typescript
addDailyResult(150, "Ваше имя");
```

#### `getCurrentPlayerRank(): DailyLeaderboardEntry | null`
Получить позицию текущего игрока в рейтинге (или null если не в топ-10).

```typescript
const myRank = getCurrentPlayerRank();
if (myRank) {
  console.log(`Вы на месте #${myRank.rank}`);
}
```

#### `getTimeUntilReset(): { hours: number, minutes: number, seconds: number }`
Получить время до следующего сброса рейтинга.

```typescript
const time = getTimeUntilReset();
console.log(`Сброс через ${time.hours}:${time.minutes}:${time.seconds}`);
```

#### `getOrCreatePlayerId(): string`
Получить или создать уникальный ID текущего браузера.

```typescript
const playerId = getOrCreatePlayerId();
```

#### `setPlayerName(name: string): void`
Установить имя для текущего игрока.

```typescript
setPlayerName("Мой никнейм");
```

#### `getPlayerName(): string`
Получить имя текущего игрока (или "Аноним").

```typescript
const name = getPlayerName(); // "Мой никнейм"
```

## 🎯 Игровой цикл

1. **Игрок открывает приложение**
   - При загрузке система проверяет, нужен ли сброс рейтинга
   - Если наступил новый день → рейтинг очищается

2. **Игрок проходит викторину**
   - Отвечает на 10 вопросов
   - Получает очки в зависимости от ответов

3. **Викторина завершена**
   - На экране ResultScreen автоматически добавляется результат
   - Игроку показывается зеленое уведомление
   - Показывается сколько очков заработано

4. **Игрок идет на таблицу лидеров**
   - Видит топ-10 за сегодня
   - Видит свою позицию (если в топ-10)
   - Может установить имя для отображения
   - Видит время до сброса рейтинга

5. **Следующий день (полночь UTC)**
   - Рейтинг автоматически очищается
   - Все начинают с нулевых очков
   - Создается новая конкуренция

## 🔮 Готово к наградам

Система полностью подготовлена для добавления наград:

### Возможные награды:
1. **За место в рейтинге**
   - 🥇 1-е место: +50 бонус-очков
   - 🥈 2-е место: +30 бонус-очков
   - 🥉 3-е место: +20 бонус-очков

2. **За количество попыток**
   - 10+ попыток в день: Медаль "Марафонист"
   - 5+ правильных ответов подряд: Бонус +25 XP

3. **Ежедневные челленджи**
   - Ежедневная цель: 500 очков = Бонус XP
   - Еженедельный лидер: Специальный статус

### Как добавить награды:

```typescript
// В LeaderboardScreen или создать отдельный компонент
const checkAchievements = (entry: DailyLeaderboardEntry) => {
  if (entry.rank === 1) {
    // Дать награду за 1-е место
    rewardPlayer('FIRST_PLACE', 50);
  }
  
  if (entry.attemptsCount >= 10) {
    // Дать медаль за марафон
    rewardPlayer('MARATHON', 25);
  }
};
```
