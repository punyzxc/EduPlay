## 📁 Новые файлы

### 1. **`src/utils/dailyLeaderboard.ts`** (277 строк)
Основной модуль с логикой системы:
- `getDailyLeaderboard()` - получить топ-10
- `addDailyResult()` - добавить результат
- `getCurrentPlayerRank()` - получить позицию игрока
- `getTimeUntilReset()` - время до сброса
- `getOrCreatePlayerId()` - уникальный ID браузера
- `setPlayerName()` / `getPlayerName()` - управление именем
- `getLeaderboardDateFormatted()` - красивая дата
- `clearDailyLeaderboard()` - очистить (отладка)

**Особенности:**
- Все данные в localStorage
- Автоматическая проверка и сброс при смене дня
- Группировка результатов по игроку
- Сортировка по очкам
- Ограничение топ-10
